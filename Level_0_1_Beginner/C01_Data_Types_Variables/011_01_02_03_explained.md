## Solution Approaches

### Approach 1: Direct / Straightforward
```cpp
#include <iostream>
#include <iomanip>
int main() {
    // Why 0.1 + 0.2 != 0.3 in floating point
    double a = 0.1;
    double b = 0.2;
    double c = a + b;
    
    std::cout << std::setprecision(20);
    std::cout << "0.1 = " << a << "";
    std::cout << "0.2 = " << b << "";
    std::cout << "0.1 + 0.2 = " << c << "";
    std::cout << "0.3 = " << 0.3 << "";
    std::cout << "Equal? " << (c == 0.3) << "";  // false!
    
    // Binary representation cannot store 0.1 exactly
    // 0.1 in binary = 0.0001100110011... (repeating)
    return 0;
}
```
